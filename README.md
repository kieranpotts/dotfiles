# Dotfiles

**A baseline, extensible user profile configuration for UNIX-like systems (including Git Bash for Windows).**

Features include:

- Better Git defaults
- Extensive suite of useful Git aliases
- Global `.gitignore` file
- Customized Bash prompt
- Handy UNIX aliases
